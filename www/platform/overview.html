Measurment Platform

Our measurements run in three phases:
  • Phase I: Workers on PlanetLab probe the top 1 million Alexa sites, using NPN to determine which sites announce HTTP/2 support. (Runs daily.)
  • Phase II: Workers in our labs in Cleveland, USA and Barcelona, Spain attempt to fetch the root object for each site that claims to support HTTP/2. The result is a list of sites that actually support HTTP/2. (Runs daily.)
  • Phase III: One worker in Barcelona, Spain fetches each site that actually supports HTTP/2 using HTTP/1.1, SPDY, and HTTP/2 and records performance and usage information like page load time and number of TCP connections used. (Runs weekly.)
