Page Differences

Web pages are made up of multiple objects. This first object fetched is an HTML document, which tells your browser which additional objects it should load and how to display them. Additional objects could include scripts, style sheets, and images.

For complex pages, these objects are often served from different servers. For example, a site might host all of its HTML and JavaScript on one server and its images on another. Sometimes, sites serve pages differently over HTTP 1 and HTTP 2:

These results compare the HTTP 1 and HTTP 2 versions of the sites we crawled in terms of:

  • Number of Objects: How many objects did each version of the page load?
  • Number of TCP Connections: How many connections did our browser make when it loaded each version of the page?
  • Number of Domains: How many servers did our browser contact when it loaded each version of the page?
