FAQs

NOTE: These are placeholder FAQs from my HTTPS dashboard site.

Which URLs do you crawl?

Before each crawl, we download the most recent version of the global Alexa Top 500 Sites. In the future, we hope to expand this list.

When and where do you perform crawls?

Our nightly crawls are performed from Carnegie Mellon University in Pittsburgh, PA, USA. In the future, we hope to crawl from multiple locations.

What software do you use?

We capture a HAR for each page using Google Chrome and Chrome HAR Capturer. We generate screenshots using PhantomJS.

Our Web profiling code is available on Github.

Can I see a side-by-side comparison of a site's mobile and desktop versions?

Sorry, not right now.

Can I see how a site's structure has evolved over time?

Not right now, but we're working on it.

It looks like the screenshot doesn't completely match the numbers in the profile. Am I crazy?

This is possible. Our profiles are based on HARs captured by Chrome, but we haven't found a good way to render a page to an image file from Chrome via the command line (if you have suggestions, we're all ears!). So, for the time being, we're making a second pass using PhantomJS to capture screenshots. If Chrome and PhantomJS load a page differently (e.g., sometimes Chrome refuses to load mixed content), there may be discrepancies between the numbers in the profile and the screenshot.

Some of the screenshot links are broken. What gives?

Due to space constraints, we only save full-size screenshots for the past week. Thumbnails are kept for all crawls.
